# README #

Documentación sobre Webnu

### DETALLES ###

* Webnu permite subir la carta de un restaurante, de manera dinámica desde un área de cliente
* Genera una URL publica https://webnu.es/carta/<nombre-restaurante>.
* También pueden seleccionar un PDF y tener una carta con flipbook.
* Desde el área de cliente, el restaurante puede exportar su QR para imprimir.
* Al registrarse desde el front, se creará el cliente y la correspondiente suscripción (mensual o anual) en Stripe.

### IMPORTANTE ###

* Al realizar una subida, no reemplazar el archivo .env ni el /public/.htaccess
